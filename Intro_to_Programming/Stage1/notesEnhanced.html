Stage 1: Make a Web Page

Google Chrome - top right - three dot icon, More Tools, Developer Tools to see html code behind a website

  1. HTML and CSS are both "languages"
  2. HTML controls the "structure" of a web page
  3. CSS controls the "style" of a page (how it looks).
  4. When programmers talk about the "DOM" (Document Object Model) they are talking about the tree-like structure of a page.
  • HTML - HyperText Markup Language - the standard markup language used to create web pages.
  • CSS - Cascading Style Sheets - style sheet language used for describing the look and formatting of a document written in a markup language.
  • DOM - Document Object Model - a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages). The nodes of every document are organized in a tree structure, called the DOM tree.

Boxes everywhere - everything on a webpage is a box - there's also a grid in the background to make sure everything stays aligned. Start with the big boxes and work down from there, draw it out. Use div tag to create boxes
