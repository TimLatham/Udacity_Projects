About CSS

CSS is used to keep the design away from the document. It also allows you to follow the coder credo, "never repeat yourself". By integrating the use of structures such as div tags and CSS you can easily control formatting across a variety of elements with minimal lines of code/repetition.

For example, if you name your div class "section", you can format all div class "section" with "".section {"css code here"}"". This is called a class selector.

Class Selectors

Class selectors are your friend. They are probably the most useful and versatile selectors out there. In part because they are well supported in all browsers. In part because you can add multiple classes (just separated by a space) on HTML elements. In part because there are JavaScript things you can do specifically for manipulating classes.
